🧠 OllamaBot Kids — Parle à une IA locale

Tutoriel d'enregistrement d'écran pour OllamaBot Kids. De l'installation à la conversation avec une IA qui tourne sur ton propre ordinateur.

3:30Durée
7Étapes
ExplorateurNiveau
🖥️Enreg. écran
🎬 Script d'enregistrement
1
C'est quoi OllamaBot ?
0:00–0:15
Action : Montrer l'écran d'intro expliquant l'IA locale
« OllamaBot fait tourner une IA sur ton propre ordinateur. Pas de cloud. Aucune donnée envoyée nulle part. Tes conversations restent privées. »
2
Installer Ollama
0:15–0:45
Action : Montrer ollama.com → télécharger → installer → terminal : ollama run tinyllama
« Télécharge Ollama depuis le site. Une seule installation. Puis dans le terminal, lance ollama pull tinyllama — un petit modèle parfait pour les enfants. »
3
Ouvrir l'appli et se connecter
0:45–1:10
Action : Ouvrir OllamaBot Kids → cliquer sur Connecter → indicateur vert
« Ouvre l'appli, clique sur Connecter. Vert signifie que ton IA locale tourne et est prête. »
4
Première conversation
1:10–1:50
Action : Taper « Explique l'informatique quantique à un enfant de 10 ans » → montrer la réponse en streaming
« Demande-lui ce que tu veux. Regarde la réponse arriver en temps réel. Cette IA tourne sur TA machine — pas dans un data center. »
5
Modes de personnalité
1:50–2:30
Action : Basculer entre les modes Pirate, Scientifique, Comique, Professeur
« Quatre modes de personnalité. L'IA devient un pirate, un scientifique, un comique ou un professeur. Même question, réponses complètement différentes. »
6
Dev Lab
2:30–3:10
Action : Aller dans Dev Lab → montrer l'éditeur de prompt système → ajuster le curseur de température
« Dev Lab te permet d'écrire des prompts système personnalisés et d'ajuster les paramètres. La température contrôle la créativité — bas c'est factuel, haut c'est délirant. »
7
Essayer d'autres modèles
3:10–3:30
Action : Montrer le sélecteur de modèle → basculer entre tinyllama, phi, gemma
« Change de modèle quand tu veux. Les petits modèles sont plus rapides, les gros sont plus intelligents. Expérimente et trouve ton préféré. Ça, c'est OllamaBot Kids — une vraie IA, locale, faite pour apprendre. »
