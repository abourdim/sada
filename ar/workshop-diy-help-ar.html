🔧 TROUBLESHOOTING

Fonts look wrong / different
Google Fonts need internet on first load. After caching, they work offline. If no internet, system fonts are used as fallback — functional but less pretty.
Animations are laggy
Some files use heavy CSS/JS animations. Close other browser tabs. Try Chrome (best performance). Hardware acceleration should be enabled in browser settings.
File won't open / shows code
Make sure you're opening the .html file in a browser, not a text editor. Right-click → Open With → Chrome/Firefox. Don't open raw files from inside a ZIP — extract first.
Interactive elements don't work
JavaScript must be enabled (it is by default). Some files need mouse/touch interaction — look for clickable elements, buttons, or hover areas. Try clicking the main visual element.
File looks cut off on mobile
Most files are responsive but some complex layouts (wiring diagrams, comparisons) are desktop-optimized. Rotate to landscape or view on desktop for best experience.
Bug #42 appeared in my file
This is expected behavior. Bug #42 cannot be removed. It has been legally reclassified as a Feature. Do not attempt deletion. Do not file a report. Check the time — it's probably 3:42 AM.
