🛠️ Technical Reference

Architecture

Every file is a single self-contained HTML document. No build step. No bundler. No framework. Inline CSS and JavaScript. The only external dependency is Google Fonts loaded via CDN — everything works offline once fonts are cached.

File Naming Convention

workshop-diy-[format-name].html

All lowercase, hyphens for spaces. The prefix workshop-diy- ensures all files sort together in any directory.

Design System

Files share a common aesthetic: dark backgrounds (#0a0a14), JetBrains Mono for code, Syne for headers, Inter for body text. Accent color is Workshop orange (#ff6600). But each file adopts the visual language of whatever it's mimicking — Tinder looks like Tinder, newspapers look like newspapers.

❓ Frequently Asked Questions

Do I need to install anything?

No. Open any .html file in a browser. That's it.

Can I use these offline?

Yes. The only online requirement is Google Fonts on first load. After that, fonts are cached and everything works offline.
