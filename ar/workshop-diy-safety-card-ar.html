1. Alert instructor immediately. 2. Reduce code deployment rate by 50%. 3. Switch to emergency tea reserves. 4. Do NOT attempt complex debugging. 5. Await resupply.

🐛 BUG DISCOVERED

1. Do not file a bug report. 2. Observe the bug for 30 seconds. 3. If it produces unexpected joy, reclassify as feature. 4. If it produces unexpected chaos, also reclassify as feature. 5. All أخطاء are ميزات.

🤖 ROBOT TABLE DROP

1. Do not panic. The robot has survived 47 previous drops. 2. Check wheels, sensors, and battery compartment. 3. If operational, continue session. 4. If not, the robot earned an honorable retirement.
