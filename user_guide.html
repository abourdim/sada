What is صدى sada

صدى (sada) means "echo" in Arabic. It's a collection of 46 interactive HTML apps built for kids learning robotics, AI, cybersecurity, and creative coding at Workshop-DIY, a Loi 1901 association in Chelles, 77500, France.

Every app is a single HTML file — no server, no install, no npm, no build step. Just open in a browser. Works offline. Works on any device.

The project exists in 3 languages: English, French, and Arabic (with full RTL support and bismillah). It includes 46 apps presented in 164 different formats, 30 tutorial storyboards, and a complete video production kit.

صدى sada is 100% open source on GitHub, and the apps are available as a free landing page at abourdim.github.io/sada.

08

How to open & use

1
Unzip the package
Extract sada-v1.0.zip to any folder on your computer.
2
Open start_here.html
Double-click start_here.html in the root folder. It opens in your browser and links to everything.
3
Browse apps
Click any app name to open it. Each is a self-contained HTML file. Works in Chrome, Firefox, Safari, Edge.
4
Switch languages
Same app, different folder: root = English, fr/ = French, ar/ = Arabic.
5
Works offline
No internet required. No server. No install. Just HTML + CSS + JS in one file.

Requirements: A modern web browser. That's it. Chrome recommended for best experience.

For educators & workshops

In-class presentations
Open demo.html or workshop-diy-hacker-presentation.html on a projector. Auto-plays cinematic scenes. Press arrow keys to navigate.

Let students explore: CTF Quiz (cybersecurity), Escape Room (logic puzzles), Retro Arcade (retro computing), Pokédex (app discovery). All work on school Chromebooks.

Workshop displays
Use Ambient Screen, Boot Sequence, or Loading Screens on lobby TVs. Double-click for fullscreen.
D
Printables
Open Sticker Sheet, Blueprints, or Workshop Bingo → screenshot → print. Great for events.
E
Video tutorials
Follow the 10 tutorial storyboards in video/tutorials/ to film step-by-step YouTube videos. Each has scene-by-scene instructions.

Facebook Calendar (84 posts)

1
Open the calendar
Open facebook-calendar.html in your browser.
2
Pick a week
Click S1–S12 buttons to filter by week, or use the search bar.

Copy a post
Click 📋 Copier on any post card. The text is copied to your clipboard, including the app link and workshop-diy.org.

Paste to Facebook
Go to your Facebook page → new post → paste. The link auto-generates a preview card.

Add a screenshot

12

GitHub Pages deployment

The zip includes a .git folder. Push directly to GitHub and enable Pages:

# Unzip (the .git folder is included) $ unzip sada-v1.0.zip -d sada $ cd sada # Add your remote $ git remote add origin git@github.com:abourdim/sada.git $ git push -u origin main # Enable GitHub Pages # Go to repo → Settings → Pages → Source: main → Save # Your site is live at: https://abourdim.github.io/sada/

All demo links in the Facebook posts point to https://abourdim.github.io/sada/. As soon as you push, every link works.
