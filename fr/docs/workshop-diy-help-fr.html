🔧 DÉPANNAGE

Les polices semblent incorrectes / différentes
Google Fonts nécessite internet au premier chargement. Après mise en cache, elles fonctionnent hors ligne. Sans internet, les polices système sont utilisées en remplacement — fonctionnel mais moins joli.
Les animations sont saccadées
Certains fichiers utilisent des animations CSS/JS lourdes. Fermez les autres onglets du navigateur. Essayez Chrome (meilleures performances). L'accélération matérielle doit être activée dans les paramètres du navigateur.
Le fichier ne s'ouvre pas / affiche du code
Assurez-vous d'ouvrir le fichier .html dans un navigateur, pas un éditeur de texte. Clic droit → Ouvrir avec → Chrome/Firefox. N'ouvrez pas les fichiers bruts depuis un ZIP — extrayez d'abord.
Les éléments interactifs ne fonctionnent pas
JavaScript doit être activé (il l'est par défaut). Certains fichiers nécessitent une interaction souris/tactile — cherchez les éléments cliquables, boutons ou zones de survol. Essayez de cliquer sur l'élément visuel principal.
Le fichier est coupé sur mobile
La plupart des fichiers sont responsifs mais certaines mises en page complexes (schémas de câblage, comparaisons) sont optimisées pour le bureau. Passez en mode paysage ou consultez sur ordinateur pour une meilleure expérience.
Bug #42 est apparu dans mon fichier
C'est un comportement attendu. Bug #42 ne peut pas être supprimé. Il a été légalement reclassifié en Fonctionnalité. Ne tentez pas de le supprimer. Ne déposez pas de rapport. Vérifiez l'heure — il est probablement 3h42 du matin.
