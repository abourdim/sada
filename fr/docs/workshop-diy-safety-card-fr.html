🤖 Utilisation du kit robot
🔋
CHARGE
La batterie 18650 doit être chargée. Un robot mort est un robot triste.
📱
CONNEXION
Cliquez sur « Connecter » dans l'app bit-bot. Acceptez l'invitation d'appairage BLE.
🎮
PILOTAGE
Utilisez les flèches ou le joystick. Le robot répond en 50 ms.
💃
DANSE
Si le robot danse sans qu'on lui demande, c'est une FONCTIONNALITÉ, pas un bug.
⚠️ Procédures d'urgence

🔴 DÉCONNEXION BLE

1. Restez calme. 2. Actualisez le navigateur. 3. Re-cliquez sur « Connecter ». 4. Si le robot continue de bouger de façon autonome, débranchez la batterie. 5. Déclarez comme demande de fonctionnalité.

☕ PÉNURIE DE CAFÉ

1. Alertez l'instructeur immédiatement. 2. Réduisez le taux de déploiement de code de 50%. 3. Passez aux réserves de thé d'urgence. 4. NE TENTEZ PAS de débogage complexe. 5. Attendez le réapprovisionnement.

🐛 BUG DÉCOUVERT

1. Ne déposez pas de rapport de bug. 2. Observez le bug pendant 30 secondes. 3. S'il produit une joie inattendue, reclassifiez comme fonctionnalité. 4. S'il produit un chaos inattendu, reclassifiez aussi comme fonctionnalité. 5. Tous les bugs sont des fonctionnalités.

🤖 CHUTE DU ROBOT DE LA TABLE

1. Ne paniquez pas. Le robot a survécu à 47 chutes précédentes. 2. Vérifiez les roues, capteurs et compartiment batterie. 3. S'il est opérationnel, continuez la session. 4. Sinon, le robot a mérité une retraite honorable.
