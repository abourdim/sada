Features
Self-Contained
Every file works standalone. Only external dependency: Google Fonts. No npm, no build step, no framework.
Trilingual
567 files across English, French, and Arabic. Use the language cards above to set your preferred language globally (full RTL with Amiri font + Bismillah). Switch languages in the slideshow player.
30 Tutorial Storyboards
Screen-recording scripts for 10 apps × 3 languages. Each has step-by-step actions, voiceover text, and unique color themes.
Slideshow Player
demo.html — autoplay, shuffle, category filter, fullscreen, keyboard shortcuts (Space, arrows, F).
صدى Lore
Bug #42, coffee machine at 8%, password123, bit-bot's 47 drops, hot glue structural engineering — consistent across every file.
90+ Unique Formats
Fake newspapers, Tinder profiles, courtroom transcripts, cereal boxes, NASA dashboards, SCP reports, and much more.
⚡ Nav Pill
Every page has a discrete floating button (top-right). Click to navigate: prev/next app, master hub, index, calendar, shop, guide. Plus language switcher.
🌍 Global Language
Set your language once (from this page or any nav pill). All pages auto-redirect to the right version via localStorage.
🔒 Splash Screen
splash.html — cinematic entry with starfield, Arabic calligraphy, and password gate. Master code for admin, user code changeable.
